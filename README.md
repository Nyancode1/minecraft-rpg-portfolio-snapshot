# Minecraft RPG Server - Skript 코드 포트폴리오

마인크래프트 RPG 서버 개발 프로젝트의 Skript 코드 모음입니다.

## 프로젝트 개요

Skript 플러그인을 활용하여 마인크래프트 서버에 RPG 시스템을 구현한 프로젝트입니다.

## 주요 기능

### 경제 시스템
- **돈.sk** - 서버 내 화폐 시스템
- **수표뽑.sk** - 수표 발행/교환 시스템
- **상점.sk / 상점2(비주류).sk / 상점3(개복치).sk** - 다양한 아이템 상점
- **블랙마켓.sk** - 특수 아이템 거래소

### 장비 시스템
- **무기.sk** - 무기 시스템 기반
- **무기-강화.sk / 방어구-강화.sk** - 장비 강화 시스템
- **무기-등급업.sk / 방어구-등급업.sk** - 장비 등급 상승 시스템
- **무기계수1.sk / 무기계수2.sk** - 무기 스탯 계수 처리
- **강화.sk** - 통합 강화 시스템
- **기본템.sk** - 기본 장비 지급

### 컨텐츠 시스템
- **랜덤상자.sk / 로얄상자.sk** - 가챠/상자 시스템
- **도박-룰렛test.sk / 도박-주사위.sk** - 도박 미니게임
- **현상금.sk** - PvP 현상금 시스템
- **펫.sk** - 펫 동반 시스템

### 유저 시스템
- **전직.sk** - 클래스 전직 시스템
- **칭호시스템.sk** - 칭호 획득/장착
- **창고.sk** - 개인 창고 시스템
- **경고.sk** - 유저 경고 관리

### UI/UX
- **gui.sk** - GUI 인터페이스 관리
- **대화.sk / 대화-튜토리얼1.sk** - NPC 대화 시스템
- **채팅.sk** - 채팅 포맷/필터

### 기타
- **히든리스트.sk** - 히든 아이템/컨텐츠 목록

## 기술 스택

- **Skript** - 마인크래프트 서버 스크립팅 언어
- **Minecraft Java Edition** 서버 환경
